# 📖 likelion-guestbook

> Every visitor is a contributor.

GitHub의 Commit과 Contribution Graph에서 영감을 받아 제작한 방명록 서비스입니다.

방문자는 단순히 글을 남기는 것이 아니라 하나의 Commit을 생성하게 되며, 작성된 방명록은 Commit History에 기록되고 Contribution Graph에 반영됩니다.

---

## 🚀 Features

### Commit Guestbook
- 제목 작성
- 작성자 입력
- 내용 작성
- 비밀번호 설정

### Contribution Graph
- 방명록 작성 시 Contribution 추가
- GitHub 스타일 잔디 그래프 반영
- 연도별 활동 조회

### Commit History
- 작성된 방명록을 Commit 형태로 표시
- 최신순 정렬

### Repository Statistics
- Total Commits
- Contributors
- Current Streak
- Longest Streak
- Contribution Score

### Achievements
- First Commit
- Contributor
- Open Source Member
- Git Master
- Legendary Maintainer

---

## 💡 Concept

기존 방명록은 단순히 게시글을 남기고 끝나는 경우가 많습니다.
Guestbook Repository는 사용자의 기록을 GitHub의 활동 방식으로 재해석했습니다.
모든 방명록은 하나의 Commit이 되며, 방문자의 흔적은 Contribution Graph에 기록됩니다.
즉, 사용자는 Contributor가 되어 저장소를 함께 만들어 갑니다.
